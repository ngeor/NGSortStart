Τι κάνει το πρόγραμμα;
-------------------------
Το NGSortStart ταξινομεί το μενού 'Έναρξη' των Windows 98. Μπορεί να έχετε παρατηρήσει ότι μερικές φορές τα Windows προσθέτουν τις ομάδες των προγραμμάτων σας στο τέλος του μενού 'Προγράμματα'. Αυτό με ενοχλεί αρκετά... Θέλω να έχω όλες τις ομάδες στην κορυφή του μενού 'Προγράμματα'! Επίσης θέλω να έχω όλα τα προγράμματα μου ταξινομημένα αλφαβητικά για να μην τα ψάχνω με τις ώρες! Ξέρω ότι στα Windows 98 μπορείς να αλλάζεις τη σειρά των στοιχείων του μενού 'Έναρξη' με ένα απλό 'σύρε κι άσε' αλλά αυτό μπορεί να γίνει αρκετά βαρετό :-(
Γι' αυτό έγραψα το NGSortStart. Ταξινομεί τα πάντα στο μενού 'Έναρξη' και μου δίνει αυτό που πραγματικά θέλω!
